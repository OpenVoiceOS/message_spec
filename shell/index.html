"groupName": str		 Displays advanced configuration for a group. display_advanced_config_for_group

GUI events, sent via the gui protocol and appear in the bus with skill_id prepended, i.e. ovos_gui_plugin_shell_companion.{msg_type}

In the GUI websocket these events look like this

{
    "type": "mycroft.events.triggered",
    "namespace": "ovos_gui_plugin_shell_companion",
    "event_name": MSG_TYPE,
    "data": {}
}
